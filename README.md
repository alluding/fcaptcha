# FCaptcha
### An unofficial Python wrapper for [FCaptcha](https://fcaptcha.lol).

# Solves

- hCaptcha
- reCaptcha (`Coming soon, as they have yet to add it`)

More coming soon.

Look at `example.py` to learn how to use it.

# Installation

```py
pip install git+https://github.com/alluding/fcaptcha
```
